# bradthebeeble's Claude Code Plugins

A marketplace of Claude Code plugins.

## Installation

```bash
/plugin marketplace add bradthebeeble/claude-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [mcp-macos-cua](https://github.com/bradthebeeble/mcp-macos-cua) | macOS Computer Use Agent — GUI automation via screenshots, clicks, typing, accessibility, and AppleScript |

## Installing a plugin

After adding the marketplace:

```bash
/plugin install mcp-macos-cua@bradthebeeble-plugins
```
